## [HowIO](https://howio.world) API

个人站点的 API，用 [Go](https://go.dev) 语言进行开发的接口，前端页面放在[这里](https://github.com/feilongjump/howio.world)了

### 目录结构

```
.
├── .air.toml   自动重载应用配置文件
├── .gitignore  git 忽略文件
├── go.mod      Go 模块文件
├── go.sum      Go 模块的依赖版本文件
├── main.go     项目应用入口文件
└── README.md   项目手册
```
